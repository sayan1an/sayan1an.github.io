  3. Pathtracer

    4. I implemented this Pathtracer as part of Image Synthesis course. The tracer supports Explicit Path Tracing, BVH accelerator, BRDFs like GGX, Beckmann, Phong(Microfacet), Phong(Classic), BRDF sampling for all the mentioned BRDFs, Sphere and Area light sampling, Environment map sampling and Multiple Importance Sampling.   4. John-The-Ripper

    5. John-The-Ripper is an open-source password security auditing tool used by system administrators for recovering lost passwords. My main contributions were mask mode password generation on GPU and a lookup table- which requires exactly two memory access for checking cryptography hashes on GPU. My work speeds up the hash rate by a factor of 100x-1000x for fast hashes. Previously passwords were generated on CPU which had to be transferred to the GPU for processing and then fetched back from GPU for checking. Now both generation and checking are done on GPUs eliminating the latencies involved in the process.

Google Summer of Code, 2013

Mentoring Organisation: Openwall
June 17 - Sept 27
Project: Implemented mask mode password generation on GPUs for John-The-Ripper security auditing tool. A mask is a blueprint(specified by the user) according to which character passwords are generated which are then encrypted and matched against the hashes that needs to be recovered. Generating data on GPU is much more crucial for fast cryptography hash types where data transfer overhead consumes majority of compute time. Combined with hash checking on GPUs, which I implemented outside GSoC, resulted in more than 300x speedup for fast hash types.
